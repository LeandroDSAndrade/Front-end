Projeto: Criação de site voltado para área de saúde

O projeto pertence ao desafio do módulo II do curso de formação em HTML Web Developer da DIO.

Consiste na criação de um site para um cliente fictício que atua do ramo de saúde (especialidade opção do desenvolvedor). Neste site devemos utilizar as tags HTML de estrutura, tabela, formulário e mídias até então estudadas.

O objeto de avaliação do desafio é o racíocínio do dev em estruturar seu código HTML conforme os requisitos solicitados, sem se preocupar com a parte estética (que fica a cargo do CSS tema de outro curso).

O site consiste em 4 páginas navagáveis entre si: Página principal, sobre, Horários de atendimento e contatos.

Para cumprir o desafio, além das tags aprendidas, busquei sobre as propriedades CSS para tornar o site bem estruturado e esteticamente bonito. Utilizei o Google AI Studio para gerar uma logomarca para a clínica e a ferramenta Photoshop para tratar a mesma. Também utilizei a IA para gerar textos bases para o conteúdo do site.
